# ☕ GetMEAChai – Creator Crowdfunding Platform

**GetMEAChai** is a full-stack, fully responsive **crowdfunding platform** that allows **fans to support their favorite creators** by sending payments. Inspired by platforms like "Buy Me a Coffee", this project integrates Razorpay for smooth payments and is built using **Next.js**, **Tailwind CSS**, and **MongoDB**.

---

## ✨ Features

- 👥 Fans can support creators directly
- 💳 Razorpay payment gateway integration
- 🧾 Store creator support data in MongoDB
- ⚡ Built with Next.js App Router (Full-stack React)
- 🎨 Responsive, modern UI using Tailwind CSS
- 📱 Mobile-first responsive design
- 🔐 Secure transaction flow (frontend + backend logic)

---
